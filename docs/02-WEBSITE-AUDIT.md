# Wise Homeopathy Website Audit
## Version 2.0

> Audit framework for refining the existing Wise Homeopathy website.

## Purpose

This document guides a structured review of every page before any implementation.

The objective is to improve the current website—not rebuild it.

---

# Audit Rules

Every page and every major section must be marked as one of:

- 🟢 KEEP
- 🟡 IMPROVE
- 🔴 REMOVE
- 🔵 ADD

Every recommendation must explain **why** it is needed.

---

# Review Criteria

## User Experience
- Navigation
- Readability
- Mobile usability
- CTA placement
- User flow

## Design
- Consistency
- Visual hierarchy
- Spacing
- Typography
- Component reuse

## Content
- Remove duplication
- Reduce long paragraphs
- Improve clarity
- Improve trust

## SEO
- Titles
- Meta descriptions
- Canonicals
- Structured data
- Internal linking
- Sitemap
- Broken links

## Accessibility
- Alt text
- Keyboard navigation
- Color contrast
- Semantic HTML

## Performance
- Image optimization
- Lazy loading
- CSS/JS cleanup
- Lighthouse improvements

Check for:

- Broken links

- Missing images

# Conversion Audit

Evaluate every page for:

- Booking flow

- Clear next action

---

# Deliverables

For every page produce:

- Rating /10
- Summary
- Findings
- KEEP / IMPROVE / REMOVE / ADD table
- Priority
- Recommended implementation batch

---

# Success Criteria

The audit is complete when:

- Every page has been reviewed.
- Every issue has a priority.
- No recommendation exists without a reason.
- A clear implementation roadmap is ready.
