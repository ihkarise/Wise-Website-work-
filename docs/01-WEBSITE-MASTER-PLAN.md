# Wise Homeopathy Website Master Plan
## Version 2.0

> Master blueprint for the Wise Homeopathy website.

## Vision

Create a premium healthcare website that begins a continuous digital health journey.

## Mission

1. Introduce Wise Homeopathy.
2. Build trust.
3. Educate patients.
4. Generate consultation bookings.
5. Transition patients into **My Health Journey**.

## Core Principles

- Refine, don't rebuild.
- Fast before flashy.
- Human before technical.
- One page, one purpose.
- Every page leads to the next step.

## Website Architecture

### Phase 1
Visitor → Trust → Consultation → Patient

### Phase 2
Patient Login → My Health Journey → Personal Care Plan → Wise Digital Twin → Symptom Tracker → Follow-up

## Primary Navigation

- Home
- Conditions
- Online Consultation
- Doctors
- Resources
- Contact

Patient Login remains separate.

## Page Goals

- Home: Why choose Wise?
- Conditions: Can you help my condition?
- Online Consultation: How does it work?
- Doctors: Can I trust you?
- Resources: Educate me.
- Contact: How do I book?

## UX Principles

- Simple navigation
- Strong internal linking
- Floating section guide
- Short readable paragraphs
- Mobile-first

## Technology

- CSS
- JavaScript
- Google Apps Script
- Google Sheets
- Netlify
- OpenRouter (future)

## Roadmap

### Phase 1

## Guiding Principle

Does this make the patient's healthcare journey simpler, clearer, calmer, and more trustworthy?
