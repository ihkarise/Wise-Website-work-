# Wise Homeopathy SEO & GEO Standards
## Version 2.0

> SEO, AEO and GEO standards for every page.

# Objectives

- Rank well in search engines.
- Be understandable by AI search systems.
- Build topical authority.

---

# Every Page Must Include

- Unique title
- Unique meta description
- Canonical URL
- Open Graph tags
- Structured data
- Internal links
- FAQ
- Medical disclaimer (where appropriate)

---

# URL Rules

- Short
- Descriptive
- Lowercase
- Hyphen-separated

---

# Internal Linking

Every page should link to:
- Related conditions
- Related articles
- Doctors
- Consultation
- Contact

No orphan pages.

---

# Schema

Use where appropriate:
- MedicalClinic
- Physician
- FAQPage
- BreadcrumbList
- Article
- BlogPosting

---

# GEO (Generative Engine Optimization)

Every page should answer:

- What is it?
- Symptoms
- Causes
- Investigations
- Treatment approach
- FAQ

Provide concise summaries before detailed content.

---

# Images

Every image:
- Optimized
- Descriptive filename
- Alt text
- Lazy loading

---

# Performance

Target Lighthouse:
- Performance 95+
- Accessibility 95+
- Best Practices 95+
- SEO 100

Underlying performance rules (image optimization, lazy loading, minimal JS, caching):

# Technical SEO

Maintain:
- sitemap.xml
- robots.txt
- Canonicals
- Structured data
- No broken links
- HTTPS

# Ongoing Review

Review quarterly for:
- Broken links
- Metadata
- Rankings
- New content opportunities
