# Wise Homeopathy Design System
## Version 2.0

> This document defines the complete visual language, interaction principles, UI standards, and UX philosophy for the Wise Homeopathy website.

## Design Philosophy

Wise Homeopathy is not a hospital website.
It is not a SaaS dashboard.
It is not a startup landing page.

It should feel:
- Premium
- Medical
- Calm
- Intelligent
- Human
- Trustworthy
- Modern
- Timeless

## Core Principles

1. Clarity over Decoration
2. Trust Before Technology
3. One Goal Per Screen
4. Calm Interface
5. Human First

## Brand Personality

Inspired by:
- Apple
- Mayo Clinic
- Stripe Documentation
- Notion

Avoid:
- Generic WordPress templates
- Startup hype pages
- Hospital ERP interfaces
