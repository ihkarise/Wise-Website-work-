# Wise Homeopathy Development Standards
## Version 2.0

> Engineering standards for all future development.

## Core Principles
- Refine, don't rebuild.
- Preserve existing branding and SEO.
- Keep commits small and reversible.
- Mobile-first.
- Accessibility-first.

## Technology Stack
- HTML
- CSS
- JavaScript
- Google Apps Script
- Google Sheets
- Netlify

Do not introduce React, Vue, Next.js, Tailwind or other frameworks unless explicitly approved.

## Coding Standards
- Semantic HTML
- Reusable CSS
- Modular JavaScript
- Consistent naming
- No duplicated logic

## Performance
- Lighthouse 95+
- Lazy-load images
- Minimize JavaScript
- Optimize assets

## Before Every Commit
- Test desktop/mobile
- Check navigation
- Validate forms
- Check console errors
- Verify accessibility
- Verify SEO metadata

## Definition of Done
- Meets Design System
- Meets UX Guidelines
- Meets SEO Standards
- No regressions
- Documented in CHANGELOG
